## Com treballar amb Claude

```bash
# Obre Claude Code a l'arrel del repositori
claude

# Claude llegirà CLAUDE.md automàticament i tindrà tot el context
# Per a tasques específiques, indica la carpeta:
> "Treballa a docs/linkedin/ i genera el text de sobre_mi"
> "Treballa a web-nextjs/ i afegeix una card al portfolio"
```

## Inici ràpid — Web

```bash
cd web-nextjs
npm install
npm run dev
# → http://localhost:3000
```
